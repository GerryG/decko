This page offers a Developer's introduction to the Wagn code base. It assumes that you've experimented enough with Wagn to have some familiarity with the basics (cards, types, rules, etc.) If you have no idea what a card is, you may find it helpful to begin at Wagn.org to get a sense of what Wagn's all about.

It also assumes you're looking at it from a docs site (ie, NOT GitHub). The page is peppered with links; if they're breaking, try FIXME - give link here.

Finally, it assumes you're a ruby developer or, at least, someone who does not fear ruby code.

Overview

If you're new to Wagn, you may be trying to wrap your head around whether Wagn is an application or a development framework. It's both. When you install a new Wagn, you get a lot more out of the box than you would with a typical dev framework; you have a working application right away.
